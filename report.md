## 1. 모티베이션 & 인트로
중간발표 단계에서는 동일한 공간적 위치에서 측정 Node를 이동해가며 동시 수집한 Wi-Fi RTT와 UWB의 이기종(Heterogeneous) 무선 Dataset을 활용하여, 가중치 최소제곱법(Weighted Least Squares) 기반의 실내 측위 실험을 진행하였습니다. 당초 일반적인 전파 Domain의 기하학적 이론상으로는 초광대역 주파수를 사용하여 시간 분해능이 압도적으로 높은 UWB 신호가 Wi-Fi RTT 신호보다 훨씬 정밀한 거리를 계측해내야 정상입니다. 하지만 복잡한 기계 장비가 밀집된 실내 공장의 복잡한 다중경로 채널 환경 하에서는 이론과 정반대의 통계적 반전 현상이 관찰되었습니다. 실전 계측 결과, 오히려 Wi-Fi RTT 측정치가 공간적 굴절 Noise에 대해 상대적으로 강건하여 더 적은 오차 분포를 보였던 반면, UWB 데이터는 특정 장애물 구간에서 전파가 완전히 차단되거나 지연되면서 실제 기하학적 거리와 매우 큰 편차를 지닌 극심한 양의 오차 Bias를 다량 발생시켰습니다.

공학적으로 이러한 현상을 방어하고 고정밀 측위를 달성하기 위해서는 기기별 특성에 따라 신뢰도가 완전히 역전된 무선 환경의 고유 Profile을 인지하고, Wi-Fi RTT와 UWB 각각의 측정 성분에 독립적인 적응형 이기종 가중치(Device-specific Weights)를 차등 부여하여 오차를 감쇄시켰어야 했습니다. 하지만 당시 알고리즘은 무선 장비 고유의 물리적 채널 특성을 전혀 반영하지 못했습니다. 단지 전체 수집 데이터의 단일 MAE 지표나 기하학적 거리에만 비례하여 획일적인 가중치 행렬을 적용하는 naive한 수식에 그쳤습니다. 결과적으로 다양한 Noise가 복합적으로 유발되는 동적 환경에 유연하게 대처하지 못해 사각지대 오차가 통제 불능 상태로 확산되었으며, 가중치 최소제곱법 알고리즘 자체의 위치 추정 성능 또한 매우 저조한 수치에 머무르는 명확한 한계를 직면하였습니다.

이러한 수식 기반 알고리즘의 실패는 결국 사람이 직접 손으로 기기별 Noise 특성과 공장의 비선형적 왜곡 성분을 수학 수식으로 정의하여 규제화하는 것 자체가 불가능하다는 통계학적 인사이트를 제공하였습니다. 이에 착안하여, Wi-Fi RTT와 UWB 신호의 Scale 편차와 하드웨어 고유의 왜곡 패턴을 수천 개의 뉴런 가중치 행렬 속에 투상하여 스스로 규칙을 조율하고 최적화할 수 있는 Deep Learning 기반의 공간 회귀 알고리즘을 구상하게 되었습니다. 본 연구에서 제안하는 하이레벨 대안은 입력층 정규화와 잔차 학습 메커니즘을 결합한 심층 신경망 모델인 AdvancedResidualNet입니다. 이 모델은 무선 환경 Noise와 기기별 Bias가 뒤엉킨 18차원의 이기종 입력 Vector $X \in \mathbb{R}^{18}$을 정교하게 차원 결합하여, 하드웨어 편차가 정화된 청정 공간 상의 2차원 평면 좌표 $y \in \mathbb{R}^2$로 마이크로초 단위 만에 직접 매핑함으로써 중간발표 단계의 구조적 취약점을 완벽하게 해소합니다.

## 2. 알고리즘 설명
본 프로젝트를 해결하기 위해 독창적으로 설계하여 최종 채택한 AdvancedResidualNet 알고리즘의 구체적인 수학적 연산 흐름과 데이터 제어 단계는 다음과 같습니다. 이 명세 메커니즘을 바탕으로 추가적인 외부 설명 없이 파이토치 코드로 완벽히 재현 가능하도록 연산 차원과 수식을 정의합니다.

단계 1: 입력 데이터 배치 차원 정렬 및 동적 할당
채점 시스템으로부터 실시간으로 유입되는 매트랩 포맷 데이터의 가변적인 사용자 수 차원에 대응하기 위해, 입력 행렬의 열 크기를 런타임에 참조하여 사용자 수 $N$을 동적으로 캡처합니다. 기지국 행과 사용자 열 구조로 배열된 원천 측정 행렬 $D \in \mathbb{R}^{18 \times N}$을 심층 신경망의 미니배치 학습 규격에 부합하도록 행렬 전치(Transpose) 연산을 수행하여 최종적으로 $N$개의 유저 행과 18개의 기지국 피처 열로 구성된 입력 Tensor를 생성합니다.
$$X = D^T \in \mathbb{R}^{N \times 18}$$

단계 2: 입력 레이어 배치 정규화 (Batch Normalization)
실내 공장 환경 특성상 7미터 안팎의 단거리 신호부터 100미터 이상의 장거리 신호까지 Feature별 수치 분포의 편차가 극심합니다. 이를 완화하기 위해 신경망 전면에 18차원 Batch Normalization 레이어를 배치합니다. 미니배치 내 평균 $\mu_B$와 분산 $\sigma_B^2$을 사용하여 입력 Scale을 원점 중심으로 표준화하고, 학습 가능한 Parameter $\gamma$와 $\beta$를 통해 아키텍처 내부 경사하강법 연산의 초기 수렴 안정성을 확보합니다.
$$\hat{X} = \frac{X - \mu_B}{\sqrt{\sigma_B^2 + \epsilon}} \cdot \gamma + \beta$$

단계 3: 제1선형 결합 및 은닉층 특징 추출
표준화된 입력 Tensor $\hat{X}$는 가중치 행렬 $W_1 \in \mathbb{R}^{64 \times 18}$ 및 바이어스 Vector $b_1 \in \mathbb{R}^{64}$를 갖는 첫 번째 밀집 레이어로 유입되어 64차원의 고차원 고유 공간으로 선형 사영됩니다. 이후 내부 Batch Normalization과 비선형 활성화 함수인 ReLU를 거친 뒤, 700개라는 소량 데이터셋 환경 하에서의 무조건적인 암기(Overfitting)를 규제하기 위해 확률 0.1의 Dropout을 수행하여 최종 중간 특징 마크인 $H_1$을 도출합니다.
$$Z_1 = \text{BatchNorm}(\hat{X} W_1^T + b_1) \in \mathbb{R}^{N \times 64}$$
$$H_1 = \text{Dropout}(\text{ReLU}(Z_1), p=0.1) \in \mathbb{R}^{N \times 64}$$

단계 4: 제2선형 결합 및 Residual Skip Connection
추출된 고차원 공간 특징 $H_1$을 바탕으로 전파 왜곡 성분을 미세 조정하기 위해 가중치 행렬 $W_2 \in \mathbb{R}^{64 \times 64}$ 및 바이어스 $b_2 \in \mathbb{R}^{64}$를 지닌 두 번째 레이어 연산을 수행합니다.
$$Z_2 = \text{BatchNorm}(H_1 W_2^T + b_2) \in \mathbb{R}^{N \times 64}$$
이때 아키텍처의 핵심인 지름길(Skip Connection) 연산을 가동합니다. 1층이 대략적으로 잡아놓은 공간적 베이스라인 정보인 $H_1$을 2층의 변환 출력인 $Z_2$와 요소별로 직접 더해준 뒤 최종 ReLU 연산을 통과시켜 가중치 결합 특징 $H_{\text{res}}$를 완성합니다.
$$H_{\text{res}} = \text{ReLU}(H_1 + Z_2) \in \mathbb{R}^{N \times 64}$$
이 구조는 신경망이 매번 백그라운드가 없는 상태에서 절대적인 좌표를 무작위로 추론하는 부담을 소거하고, 다중경로 Noise로 인해 뒤틀린 미세 잔차(Residual) 성분만을 전담하여 보정하도록 유도합니다.

단계 5: 출력 레이어를 통한 2차원 공간 좌표 수축
오차가 보정된 64차원의 융합 특징 행렬 $H_{\text{res}}$는 최종 출력 레이어 가중치 $W_{\text{out}} \in \mathbb{R}^{2 \times 64}$ 및 바이어스 $b_{\text{out}} \in \mathbb{R}^2$와의 선형 결합을 통해 우리가 최종적으로 획득하고자 하는 사용자의 2차원 물리 평면 추정 좌표 행렬 $\hat{y}$로 수축 변환됩니다.
$$\hat{y} = H_{\text{res}} W_{\text{out}}^T + b_{\text{out}} \in \mathbb{R}^{N \times 2}$$

단계 6: 비용 함수 최적화 파이프라인
오프라인 훈련 루프 단계에서는 예측된 $\hat{y}$와 실제 정답 좌표 $y$ 간의 차이를 평균제곱오차(MSE) 기반 비용 함수로 설정하여 오차 역전파 미분을 수행합니다. 가중치 폭증을 규제하는 L2 정규화 페널티(Weight Decay = $10^{-4}$)를 Adam 옵티마이저에 주입하고, 학습률의 보폭을 초기 0.02에서 최종 0.0005까지 코사인 곡선을 따라 부드럽게 감쇄시키는 Cosine Annealing Scheduler를 연동하여 정답 반경의 전역 최적점(Global Minima)으로 뉴런 가중치들을 조밀하게 안착시킵니다.

## 3. Agent AI 활용 방안
본 프로젝트의 탐색적 설계 및 가중치 최적화 빌드 단계에서 대화형 인공지능 모델인 Gemini 1.5 Pro를 적재적소에 프롬프트 엔지니어링하여 협업 도구로 활용하였습니다.

AI의 주요 역할은 소량 데이터셋 제약 하에서 성능을 고도화할 수 있는 교차 검증 방법론을 브레인스토밍하고, 단일 지표인 평균 오차 분석이 빠질 수 있는 통계학적 오류를 파악하는 대조 프레임워크 설계 도구로 활용되었습니다. 특히 훈련 과정에서 일반적인 MSE 손실 함수 대신 아웃라이어에 강건한 Huber Loss 함수를 도입하는 실험을 구상할 때, 전체 MAE와 최대 오차는 다소 증가하지만 상위 5% 극단치 데이터의 가중치 왜곡을 통제하여 95% 신뢰 구간 범위를 콤팩트하게 정렬해낼 수 있다는 손실 함수의 수학적 가중치 분배 메커니즘 관점의 인과관계를 짚어내고 디버깅 메시지를 해석해 주는 역할을 수행했습니다.

본인의 역할은 AI가 제시하는 다양한 코드 템플릿이나 레이어 수치들을 무비판적으로 수용하지 않고, 실내 공장 채널의 물리학적 특성과 조교님이 제시한 실행 시간 제약 조건에 부합하는지 엄격하게 검증하는 아키텍처 감독관 역할을 맡았습니다. 특히 소량 데이터셋 환경에서 무작위 분할 학습이 유발할 수 있는 집단별 편차 리스크를 인지하고, 실험 단계에서는 데이터 분할 시드를 고정하여 공정하게 대조하되, 최종 배포용 model.pt 가중치를 구울 때는 700명의 데이터를 쪼개지 않고 전량 학습에 과감히 투입시켜 일반화 신뢰도를 극대화하는 최종 엔지니어링 의사결정을 주도하였습니다.

### 베이스라인 비교의 공정성 (Fairness) 비판
본 실험에서 제안한 딥러닝 기반 신경망 모델들을 중간발표 단계의 전통적 가중치 최소제곱법(Scenario A)과 일대일로 단순 정량 대조하는 것은 학술적인 관점에서 다소 불공정(Unfair)한 비교일 수 있습니다. 수식 기반 알고리즘은 주어진 단일 샘플의 기하학적 제약 조건만을 즉각적으로 연산하는 파라메트릭 최적화인 반면, 심층 신경망은 700명 전체 공간의 바이어스 분포 분포도를 통째로 학습하여 공간의 빈틈을 매끄럽게 보간하는 넌파라메트릭 범용 함수 근사기(Universal Function Approximator)로 작동하기 때문입니다. 그럼에도 불구하고, 물리 수식만으로는 절대 풀 수 없었던 실내 공간의 전파 지연 한계 오차가 수치적으로 몇 미터나 존재하는지 그 기하학적 베이스라인 경계선을 명확히 계측하기 위한 비교 대조군으로서는 매우 공정하고 필수적인 절차였다고 판단됩니다.

### 알고리즘의 장점 및 단점
* 장점: 연산 효율성과 초고속 실시간성입니다. 무수히 많은 반복 미분 루프를 돌며 수 초의 지연을 유발하는 수치해석 솔버와 달리, 이미 최적화된 행렬 가중치에 입력을 통과시키는 단 한 번의 forward pass만 수행하므로 200명 추론에 단 0.000511초(인당 0.0025ms)라는 스코어보드 압도적 1위의 가속 성능을 보장합니다. 동시에 MAE를 7.9464m까지 안정적으로 통제해냅니다.
* 단점: 철저한 데이터 종속성입니다. 700개의 협소한 데이터 풀 안에서만 왜곡 패턴을 배웠기 때문에, 만약 미공개 테스트셋 300명의 사용자가 기존에 학습하지 못한 공장 외곽의 극단적인 물리적 사각지대에 상주하고 있다면 기하학적 제약 조건이 없는 신경망 특성상 쌩뚱맞은 위치로 좌표 추정치가 튀어버리는 외삽(Extrapolation) 취약점이 존재합니다.

### 자체 평가 방식의 공정성
실험 2번 파트에서 도출한 Ablation Study 표의 공정성을 확보하기 위해, 검증 Dataset 분할 모듈의 랜덤 시드를 `random_state=42`로 철저하게 동결 제어하였습니다. 하이퍼파라미터나 레이어 구조를 변경하더라도 오직 동일한 유저 집단 환경에서만 오차 지표가 계측되도록 통제함으로써, 특정 실험 케이스에 유리한 샘플이 편입되어 수치가 왜곡될 여지를 원천 차단한 공정한 통계 평가 방식임을 증명합니다.
